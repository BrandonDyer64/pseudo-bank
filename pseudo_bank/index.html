PseudoBank

A binary executable to handle a spreadsheet of transactions

Documentation

The best way to view the documentation for this program is to use cargo doc

An online version of the documentation can be found here.

Running

cargo run input-file.csv

This will process the input file and output the final account balances

Errors are output to standard error output. This application should run quickly when there aren't too many transactions that it needs to print due to error.

Example Input

type,       client, tx, amount
deposit,    1,      1,  1.0
deposit,    2,      2,  2.0
deposit,    1,      3,  2.0
withdraw,   1,      4,  1.5
withdraw,   2,      5,  3.0
dispute,    1,      1,
resolve,    1,      1,
dispute,    1,      1,
chargeback, 1,      1,

Example Output

client,available,held,total,locked
1,0.5,0.0000,0.5,true
2,2,0.0000,2,false
